# Task 5 - Contact Book

## Description
This project is a Contact Book application developed in Python. It allows users to store and manage contact information, including names, phone numbers, email addresses, and addresses.

## Features
- Add new contacts
- View all contacts
- Search contacts by name or phone number
- Update contact details
- Delete contacts
- User-friendly menu-driven interface

## Technologies Used
- Python 3
- VS Code

## How to Run

1. Save the file as `contact_book.py`
2. Open the terminal in VS Code.
3. Run:

```bash
python contact_book.py
```
