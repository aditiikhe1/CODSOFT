# Task 2 - Password Generator

## Description
This project is a Password Generator application developed in Python. It generates strong and random passwords based on the length specified by the user.

## Features
- User can choose the password length.
- Generates random passwords instantly.
- Includes:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Special characters (!, @, #, $, etc.)
- Simple and easy-to-use interface.

- Python 3
