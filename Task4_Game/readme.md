# Task 4 - Rock Paper Scissors Game

## Description
This project is a Rock-Paper-Scissors game developed in Python. The user plays against the computer, which makes a random choice each round. The game determines the winner based on standard Rock-Paper-Scissors rules and keeps track of scores.

## Features
- User chooses rock, paper, or scissors
- Computer generates a random choice
- Determines the winner automatically
- Score tracking for multiple rounds
- Play again option
- User-friendly interface

## Technologies Used
- Python 3
- VS Code

## How to Run

1. Save the file as `rock_paper_scissors.py`.
2. Open the terminal in VS Code.
3. Run:

```bash
python rock_paper_scissors.py
```
